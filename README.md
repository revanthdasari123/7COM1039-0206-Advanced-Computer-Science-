# A Machine Learning Framework for Early Detection of Depression Using Social Media Text Data from Kaggle

## Project Overview
This project develops and evaluates a machine learning framework for detecting early signs of depression from Twitter text using Natural Language Processing (NLP). The study compares Logistic Regression, Support Vector Machine (SVM), and Multinomial Naive Bayes models using TF-IDF, Bag-of-Words, sentiment, and linguistic features.

## Dataset
- Source: Kaggle – Mental Health Twitter Dataset
- Dataset link: https://www.kaggle.com/datasets/infamouscoder/mental-health-social-media
- Records: 20,000 labelled English tweets
- Classes: Depressive and Non-Depressive

## How to Run
1. Download the Kaggle dataset and place `Mental-Health-Twitter.csv` in the project folder.
2. Open the notebook (`Software.ipynb`) in Google Colab or Jupyter Notebook.
3. Install the required Python libraries.
4. Run all notebook cells sequentially.
